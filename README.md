## How to use it
1. Clone the repo (git clone https://github.com/thesecretsauce67420/Steam2DiscordRelay.git)
2. CD into the directory (cd Steam2DiscordRelay)
3. Create a file called config.json
4. Paste this text into the config (REPLACE "user" with the username of the steam account, "pass" with the password, and "webhook" with the discord webhook URL):
   ```
   {
     "user": "user",
     "pass": "pass",
     "webhook": "webhook",
   }
   ```
5. Profit!!!
